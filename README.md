# OLA Ride Insights Project

This project analyzes OLA ride data to generate insights on ride trends, revenue, cancellations, vehicle types, and customer behavior.

## Tools & Technologies
- Python (Pandas, Streamlit)
- Power BI
- Google Drive

## Project Files
- app.py – Streamlit dashboard application
- OLA_Cleaned_Data.csv – Cleaned dataset
- OLA_Ride_Insights.pbix – Power BI dashboard (shared via Google Drive)
- Ola-Slides.pptx – Project presentation

## How to Run the Streamlit App
```bash
pip install streamlit pandas
streamlit run app.py
